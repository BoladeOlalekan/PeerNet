Upload Guidelines

Admin-uploaded resources bypass the student approval workflow. They are instantly set to Approved status and will be immediately downloadable by FUTA students in the app.

  • Please double-check course selection before uploading.
  • Ensure the PDF/Slides file name is clean and descriptive.
  • For Video Lectures, upload a placeholder link or file metadata and provide a valid YouTube link.
